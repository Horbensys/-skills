## 已收录 Skills

### 集美言论识别器

好吧，这个skill是因为日常刷手机的时候真的难分辨某些言论是不是真情流露而产生的，原理如下：

识别中文互联网中以"集美"为代表的极端性别对立言论。对输入文本进行结构化分析，沿八大维度逐项评估。

- **身份归因** — 用群体身份替代逻辑推理
- **非对称普遍化** — 同一种逻辑，根据对象群体不同选择性适用
- **必胜框架** — 论点结构排除被证伪的可能性
- **语义武器化** — 将中性词汇改造为贬义标签
- **圈地排他** — 基于群体身份主张空间排斥
- **情绪-道德短路** — 主观情绪直接等同于客观道德事实
- **纯洁螺旋** — 不够极端 = 背叛，持续自我纯化
- **平台武器化** — 利用审核机制作为非对称武器

打分制，还会结合评级机制，主打对事不对人。

触发短语：集美、性别对立、小仙女言论、鉴别极端言论、分析性别话语

## 使用方式

将 `skills/` 目录下的 skill 文件夹复制到你的 Claude Code 项目的 `.claude/skills/` 中即可使用。
