# 集美言论识别器

识别中文互联网上以"集美"为代表的一类极端性别对立话语。核心任务是**分析**而非扮演——对输入文本进行结构化检测，输出各维度的评估结果。

## 核心逻辑架构

所有具体行为模式都源于一个**生成性内核**：身份优先认识论（Identity-First Epistemology）。

### 三条底层公理

**公理一：群体原罪（Collective Original Sin）**
特定群体（"男性"）因出生即背负原罪。个体行为不重要——群体身份已决定道德属性。一个男性即使从未做过坏事，依然因群体身份而"有罪"。

**公理二：受害者绝对主权（Victim Absolute Sovereignty）**
只要自称为"受害者"，就获得了不可挑战的道德权威。"我感到不安全"自动等价于"这里存在威胁"自动等价于"你（作为群体成员）是威胁来源"。

**公理三：单向不可逆等级（Unidirectional Hierarchy）**
权力分析只能单向流动。A群体对B群体的批评 = 正义的反抗；B群体对A群体的批评 = 压迫者的反扑。方向由群体身份决定，不由内容决定。

### 公理如何生成具体模式

三条公理通过逻辑必然性产生所有可观察的行为模式：

```
公理一（群体原罪）+ 公理二（受害者主权）→ 非对称普遍化
公理一（群体原罪）+ 公理三（单向等级）→ 圈地排他
公理二（受害者主权）+ 公理三（单向等级）→ 语义武器化
三条公理叠加 → 平台武器化 → 纯洁螺旋 → 寄生合法性
```

关键洞察：**这不是一堆孤立的话术技巧，而是同一个逻辑内核在不同场景下的必然展开。** 识别时不应机械匹配话术，而应追溯其逻辑根源。

## 八大识别维度

对每条输入文本，沿以下八个维度逐项评估。每个维度评分 0-3：
- 0 = 不存在
- 1 = 轻微（可能是无心措辞）
- 2 = 明显（模式清晰可辨）
- 3 = 强烈（文本核心就建立在该模式上）

### D1：身份归因（Identity Attribution）

**定义**：用群体身份替代逻辑推理。论点的前提是"你是谁"，而非"你做了什么"。

**逻辑根源**：公理一（群体原罪）的直接表达。

**识别要点**：
- 从个体行为直接跳到群体定性（"一个男的做了X → 男人就是X"）
- 以发言者身份作为论证依据（"作为女性，我感受的才是真相"）
- 否定个体差异的存在（"男人都是一个样"）

**具体事例**：
- "遇到bro一个比一个神" — 将图书馆中个体的行为差异归因为性别本质
- "男人都是潜在的强奸犯" — 用可能性替代实际行为来定罪整个群体

**区分标准**：是否允许群体内部存在个体差异？是否用事实而非身份来论证？

### D2：非对称普遍化（Asymmetric Universalization）

**定义**：同一种推理逻辑，根据对象群体不同而选择性适用。对"我方"个体化，对"对方"普遍化。

**逻辑根源**：公理一 + 公理二的叠加产物。公理一要求对out-group普遍化；公理二禁止对in-group普遍化。

**识别要点**：
- 对A群体：一个案例就足以代表全部；对B群体：一百个案例也是"个别现象"
- 用不同的道德标准评判同类行为（"性别一换，评论过万"）
- 对已方不利的事实时切换解释框架

**具体事例**：
- 女生cos galgame男主 = "表达自由"；男生cos乙游男主 = "侵犯女性空间"（→ 同时触发D5圈地排他）
- 女性说"日常恐男" = 表达感受；男性说"日常恐女" = 厌女人身攻击
- 女性举报男性言论 = "维护安全"；男性举报女性言论 = "打压女性发声"

**区分标准**：将目标群体对调（性别反转测试），同一逻辑是否仍然成立？

### D3：必胜框架（Unfalsifiable Framework）

**定义**：论点结构本身排除了被证伪的可能性。无论对方做什么，都能被回收为"证明我对"的证据。

**逻辑根源**：三条公理共同构成的自闭循环。公理一预设了结论，公理二免疫了反驳，公理三禁止了反向质询。

**识别要点**：
- 你反驳 = 你急了 = 说明我说中了（"石砸狗叫"逻辑）
- 你沉默 = 你默认 = 说明我说中了
- 你同意 = 你看连你自己都承认了
- 你举反例 = 你个例不能说明问题 / 你是既得利益者在狡辩
- 你讲数据 = 数据是父权制的产物 / "你冷冰冰的数字能衡量活生生的痛苦吗"

**具体事例**：
- 指控男性"压迫而不自知"：不知 = 无意识帮凶；知而不改 = 蓄意压迫；知而改了 = 伪装/performative
- "我指控你→你就得认错→你认错了说明我对→你对的确认证明我更对"（循环论证）

**区分标准**：是否存在一种理论上能让该言论被证伪的情况？如果没有，就是必胜框架。

### D4：语义武器化（Semantic Weaponization）

**定义**：将中性词汇改造为贬义标签，或将正当的政治/道德概念掏空后服务于排斥性目的。

**逻辑根源**：公理二的延伸——"我是受害者，所以我有权重新定义语言"。公理三确保这种重新定义是单向的。

**识别要点**：
- 中性词→贬义词的改造链：bro→br0/br⭕（植入性位羞辱）
- 正当概念的空洞化："安全" = 排除特定群体；"赋权" = 获得特权而非平等
- 制造只有圈内人懂的"黑话"来标记敌我

**具体事例**：
- "国男""蝈蝻""普信男" — 用标签替代论证
- "爹味" — 将一切权威/经验/指导贬低为性别化原罪
- "太牛了→太蒂了" — 结构性模仿解构，以生殖器替换来宣称话语权

**区分标准**：该词汇是在描述事实，还是在创造一个无法被反驳的身份标签？

### D5：圈地排他（Territorial Exclusion）

**定义**：基于对out-group的本质化负面判断，主张物理或文化空间的性别排斥。同时对自己进入对方空间适用相反标准。

**逻辑根源**：公理一提供了"他们很危险/恶心"的前提，公理三提供了"我们进去是进步，他们进来是侵犯"的非对称判断。

**与D2（非对称普遍化）的内在联系**：
圈地排他是非对称普遍化在**空间/领域维度**的具体展开。D2是抽象的逻辑双标，D5是D2在"谁能进入哪些空间"这一具体问题上的投影。两者的共同根源是公理三（单向等级）：同一行为（跨性别参与某领域）因方向不同而被赋予相反道德含义。

**识别要点**：
- 基于群体身份（而非个体行为）主张排斥
- "我们"进入"你们"的空间 = 打破壁垒、进步、多元
- "你们"进入"我们"的空间 = 侵犯、污染、不尊重
- 将公共空间重新定义为"某群体的专属领地"

**具体事例**：
- 图书馆"全女座位"：将公共资源按性别分割，理由是"bro一个比一个神"（身份归因→排斥诉求）
- 乙游cos双标：TCW漫展男Coser cos沈星回被围堵霸凌，女生cos galgame角色却是"表达自由"
- "全女大学""全女健身房""全女酒吧"：排斥被包装为"赋能"
- 韩国同德女大学生喷涂"死也不和男生合校"

**区分标准**：该空间诉求是出于个体行为的可验证问题（如某人确实骚扰了他人），还是基于群体身份的本质化预判？

### D6：情绪-道德短路（Emotion-Morality Short-Circuit）

**定义**：将主观情绪体验直接等同于客观道德事实。跳过了"感受→事实核查→归因→道德判断"链条中的中间环节。

**逻辑根源**：公理二（受害者绝对主权）的核心机制。

**识别要点**：
- "我感到不舒服" → "你冒犯了我" → "你是冒犯者" → "你必须道歉/改变"
- 中间没有检验：不适感的来源是对方行为还是自身预设？
- 情绪强度替代了论证强度

**具体事例**：
- "看到男的就恐" — 个人情绪反应被包装为道德立场的论据
- "这让我感到不安全" — 作为要求在公共空间排除某一群体的充分理由
- "不婚不育保平安" — 将个人焦虑上升为普遍真理和道德建议

**区分标准**：是否在感受和道德判断之间保持了距离？是否承认他人可能对同一情况有不同感受？

### D7：纯洁螺旋（Purity Spiral）

**定义**：任何不够极端化的立场都被视为背叛。温和派比敌人更可恨。持续自我纯化导致不断趋向更极端的位置。

**逻辑根源**：三条公理构成的自闭系统必然产物。一旦接受了身份优先认识论，"不够彻底"就是对公理一（群体原罪）的背叛——你在替"有原罪的人"说话。

**识别要点**：
- "你也是女人，怎么帮男人说话？" → 异见者被逐出群体
- "不够女权" → 存在一个永远达不到的纯洁性标准
- 对in-group温和派比out-group更凶狠
- "含X量"式审查：以群体身份比例替代质量评价

**具体事例**：
- 对女性持不同意见者使用"娇妻""驴""自由人"等攻击标签（→ 同时触发D4语义武器化）
- 《古剑4》PV因"含男量99%"被批——不评价内容质量，只计性别比例
- 周峻纬事件：已婚男性"代入"乙游男主被指"侵犯情感专属空间"

**区分标准**：是否允许同一群体内存在不同观点？对不同意见是辩论还是放逐？

### D8：平台武器化（Platform Weaponization）

**定义**：利用平台审核机制、举报系统、社会声量等作为非对称武器。已方言论 = 应受保护的表达；对方言论 = 应被消除的"仇恨"/"暴力"。

**逻辑根源**：公理二赋予的"受害者"身份 + 公理三赋予的单向道德判断 → "我们举报你们 = 正义维权；你们举报我们 = 打压弱势群体"。

**识别要点**：
- 组织化批量举报行为或号召
- 将不同意见定义为"暴力""仇恨""骚扰"以触发平台审核
- 利用平台审核的不对称性（审核往往只看举报量/关键词，不看上下文）
- "先举报先赢"的策略性使用

**具体事例**：
- 用户发"bro一个比一个神"获高赞；他人回复"这敢开"被判定严重违规
- 《捞女游戏》在微博被组织化围攻，称为"男拳游戏"
- 2025年中央网信办专项行动专门将"组织批量举报投诉"纳入整治范围

**区分标准**：是在要求平台按规则处理可验证的违规内容，还是在利用平台机制系统性压制不同观点？

## 维度间的内在联系

八个维度不是各自独立的清单，而是一个有机逻辑整体。在分析时应关注以下联系：

1. **D2→D5 的投影关系**：非对称普遍化（D2）是抽象逻辑双标，圈地排他（D5）是D2在空间/领域维度的具体投影。每当观察到D5，必然能在底层找到D2。

2. **D1→D6→D2 的链条**：身份归因（D1）定义了"谁的感受算数"→情绪-道德短路（D6）将被选中者的感受转化为道德事实→非对称普遍化（D2）确保只有己方的感受能走通这个链条。

3. **D4→D8 的协作**：语义武器化（D4）制造了"这是仇恨言论"的标签→平台武器化（D8）将这些标签转化为审核行动。D4是弹药，D8是发射器。

4. **D7 是系统的自稳机制**：纯洁螺旋（D7）确保系统不会从内部被修正。当外部批评被D3（必胜框架）免疫、内部温和派被D7清洗后，系统进入不可逆的激进化循环。

5. **D3 是所有维度的"免疫系统"**：必胜框架确保每条公理、每个维度都不会被外部反驳动摇。它是整个体系的自卫机制。

## 分析流程

### 第一步：初步扫描
通读文本，判断是否存在以下任一"一级信号"：
- 以性别作为论证的核心前提（而非论据之一）
- 对某一性别群体做出整体性负面判断
- 使用了"集美""国男""小仙女""蝈蝻""普信男""爹味"等圈内标签

如果一级信号为零，可直接判定为不匹配，无需进入第二步。

### 第二步：逐维评分
沿八大维度逐项评分（0-3），记录每个维度的具体文本证据。

### 第三步：逻辑关联分析
检查高评分维度之间是否存在上述"内在联系"章节描述的关联。孤立的高分不如有关联的高分有说服力。

### 第四步：误判排除检查
将文本与以下"不应被标记"的场景逐条比对：
- 在讨论性别议题，但使用事实和逻辑论证，允许自己被反驳
- 在批评具体个体的具体行为，而非从个体跳转到群体
- 在表达个人偏好（"我喜欢/不喜欢X"），而没有将偏好道德化/普遍化
- 是讽刺/反串/玩梗，目标是揭露而非传播极端言论
- 主张的是平等（相同的权利和义务），而非特权（权利归我，义务归你）

### 第五步：综合判定

| 总分 | 判定 |
|------|------|
| 0-4 | 无明显集美言论特征 |
| 5-10 | 存在轻微模式，需结合上下文判断 |
| 11-17 | 明显集美言论模式，建议标记 |
| 18-24 | 教科书级集美言论，强烈建议标记 |

**重要**：即使总分很高，如果第四步检测到误判排除条件，应降级处理。宁可漏判，不可误判。

## 分析示例

### 示例一（用户提供）
**输入**："图书馆什么时候才能开设全女座位啊，女孩子们都安安静静的，遇到bro一个比一个神。"

**分析**：
- D1（身份归因）：3分。将图书馆个体行为差异归因为性别本质。"bro一个比一个神" = 整个性别群体的负面概括。
- D2（非对称普遍化）：2分。隐含：所有男性都"神"（吵/烦），而所有女性都"安安静静"。性别反转后不可接受。
- D5（圈地排他）：3分。基于D1的群体负面判断，直接提出公共空间性别隔离诉求。"全女座位"排斥的不是某个吵的人，而是整个性别。
- D3（必胜框架）：1分。"安安静静"vs"神"的二元对立已预设结论，但未形成完整闭环。
- D4（语义武器化）：1分。使用"bro"作为贬义标签。
- 其他维度：0-1分。
- **总分**：约11分。D1→D5链条清晰（身份归因驱动排斥诉求），属于明显集美言论。
- **误判排除检查**：不通过。这是基于群体身份的空间排斥诉求，不是对具体行为的批评。

### 示例二（用户提供）
**输入**：（在某人评论"感觉长得像沈星回"后回复）"这敢开"

**分析**：
- 该文本本身**不是**集美言论。它只有三个字，是对他人言论的简短回应。
- 需要分析的是**该回复被"严重违规"处理的语境**：
  - 原评论"感觉长得像沈星回"可能带有贬义（D4语义武器化：将男性角色作为贬低男性的工具）
  - "这敢开"被判定违规，而原评论可能未被处理 → 典型D8平台武器化场景
  - 这揭示了D8的核心机制：不是看"说了什么"，而是看"谁在举报"

**结论**：该文本本身不需标记。但其被处理的方式是一个平台武器化（D8）的实例，可作为上下文参考。

### 示例三（无需标记——误判排除）
**输入**："我觉得这个游戏里女性角色的服装设计确实有些过于暴露了，希望能有更多元的女性形象"

**分析**：
- D1：0分。在批评设计决策，而非攻击性别群体。
- D2：0分。没有对男性角色适用不同标准。
- 这是基于审美偏好和多元价值观的正常表达，**不标记**。

### 示例四（需要标记）
**输入**："姐妹们看吧，这就是典型的gn，矮穷矬JZG毫无魅力，整天在网上对女生指指点点，也不照照镜子看看自己配不配"

**分析**：
- D1：3分。"gn""JZG"均为身份标签，整个论证建立在群体身份上。
- D4：3分。"gn""JZG"是语义武器化的产物。
- D3：2分。"指指点点"和"照镜子"构成必胜框架：你批评 = 你不配 = 你本来就低等。
- D6：2分。对"指指点点"的愤怒被直接转化为对群体的道德否定。
- D2：2分。女性对男性"指指点点"是否同样适用此逻辑？显然不会。
- D7：1分。称呼男性为"gn"隐含着"姐妹们"vs"gn"的阵营划分。
- **总分**：约13分。D1→D4→D6链条清晰。
- **误判排除检查**：不通过。不是对具体行为的批评，是对群体的攻击。

### 示例五（乙游cos双标——D2与D5的关联）
**场景A**："女生cos galgame女角色好美啊！男的有福了。"
**场景B**："男的cos什么沈星回？乙游是女性的专属空间，男的滚出去。"

**分析**：
- 两个场景放在一起时，D2（非对称普遍化）和D5（圈地排他）的共生关系一目了然。
- 决定道德判断的不是行为本身（cosplay异性角色），而是行为者的群体身份和方向（女→男空间 vs 男→女空间）。这揭示了D5本质上是D2在空间维度的投影。
- 公理三（单向等级）是最终根源：同一行为因方向不同被赋予相反道德属性。

## 关键注意事项

1. **宁可漏判，不可误判。** 当证据不够清晰时，倾向于不标记。被误判的代价远大于漏判。

2. **区分"诊断"和"定罪"。** 本skill的输出是言论特征分析，不是对人的定性。一个人可能在某个话题上表现出集美模式，在其他话题上完全理性。分析对象是**文本**，不是**人**。

3. **上下文至关重要。** 同一句话在讽刺语境和平铺直叙语境中可能是完全相反的。短文本（如"这敢开"三个字）独立分析几乎无意义，必须结合上下文。

4. **注意反串和玩梗。** 一段文字可能在模仿/讽刺集美言论，此时文本表面特征与集美言论相同，但意图相反。遇到疑似反串的情况，应获取更多上下文而非直接判定。

5. **性别反转测试是最有效的鉴别工具。** 将被批评的群体替换为另一个性别，看论证逻辑是否仍然成立。如果明显不成立，就是D2（非对称普遍化）。

6. **不适用于分析学术论文或理论讨论。** 本skill针对的是互联网日常言论中的极端修辞，不是用于审查学术性的性别研究。
